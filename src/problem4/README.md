# Sum to N

## Assumptions

### Assume A

- Supports negative input
- Example:

  ```
  sum_to_n(-5) = -5 + -4 + -3 + -2 + -1 = -15
  ```

---

### Assume B

- Only supports non-negative input
- Throws error if `n < 0`

---

## Run

Install dependencies:

```
npm install
```

Run program:

```
npm run start
```

---

## Usage

- Enter an integer when prompted
- Enter `q` to quit

Example:

```
Enter n: 5
Enter n: -5
Enter n: q
```
